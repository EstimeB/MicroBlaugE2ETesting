### About The Project
Created to perform E2E testing for [MicroBlaug App](https://github.com/EstimeB/MicroBlaug).

#### Built With
  * Java
  * Selenium
  * TestNG
  * Cucumber

### Getting Started

#### Prerequisites
   * Install Java
   * Install Java IDE
   * Maven
   * Selenium Webdriver
   * TestNG
   * Cucumber

#### Installation
  * Clone the repo
  * Open repo in an IDE

### Usage
  Can be used for/add on to, to perform types of testing such as:
  * Functional 
  * Regression 
  * Sanity 
  * Smoke 
  * UI (black box)
  * Integration 

### Roadmap
  Software Testing Life Cycle (STLC)
